# 1.44MB フロッピーシューティング企画書・仕様書

## 必要なシーン

* タイトル
* オプション
* クレジット
* ステージ選択
* ステージ

## ゲームの流れ

* 2D・3Dシューティング
    * 2Dと3Dの画面を行き来して、ステージを進めていく
    * 2Dと3Dの切り替えは、ステージの特定のポイントで行う
    * 2Dと3Dはシームレスに切り替える
    * 2Dでは、横と縦スクロールのシューティングを行う。
    * 3Dでは、自機の背後にカメラがあり、前方に敵が出現する。

## ステージ構成

* １ステージをチャプターに分ける
  * １ミスでチャプターの最初からやり直し
